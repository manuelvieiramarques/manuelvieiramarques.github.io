Enquadramento no Projeto VOICIS-CORE

  • Formações Udemy

    Numa fase inicial, foram feitas várias formações nas mais variadas ferramentas utilizadas na Itcenter. Ainda nas formações, foram também feitas formações em algumas linguagens de programação!

  • Integração na equipa de desenvolvimento

    Após definido o projeto onde iria estar inserido, fui integrado na equipa de desenvolvimento do VOICIS, com a adaptação inicial de clonar repositórios e instalar e configurar todas as ferramentas necessárias no projeto.

  • Primeiros passos dentro do Projeto

    Após feitas todas as configurações, foram dadas as primeiras explicações sobre o funcionamento do VOICIS-CORE, permitindo a exploração do mesmo tanto a nivel da interface como do código do projeto, para uma maior consolidação do funcionamento.

  • Fase de desenvolvimento

    Nesta altura do estágio, foram desempenhadas tarefas no desenvolvimento de código para novas features e/ou bugs reportados. Também foram feitos testes automaticos e outros testes para lançamentos de releases novas.
